# Data Analytics Project: Regional Analysis of the Northeast Monsoon (2015-2020)

## Executive Summary
This project analyzes the thermal signature of the Northeast Monsoon (Amihan) across a latitudinal chain from South Korea to the Philippines. Using ERA5 atmospheric data (925 hPa), I quantified the southward progression of cool surges and their impact on regional climates.

## Technical Skills Demonstrated
- **Data Engineering:** Consolidating multi-year meteorological datasets across 4 international stations.
- **Atmospheric Physics:** Analyzing lower-troposphere temperature (925 hPa) to track air mass modification.
- **Statistical Analysis:** Calculating regional correlations and thermal gradients using Python (Pandas/NumPy).
- **Visualization:** Developing climatological lifecycle plots and correlation heatmaps (Seaborn/Matplotlib).

## Key Findings
1. **Thermal Synchronization:** High correlation (>1.204 coefficient) between mid-latitude and tropical cooling cycles.
2. **Onset Gradient:** The thermal 'drop' intensity scales with latitude, losing ~12.04°C in Korea vs ~1.71°C in the Philippines.
3. **Onset Predictability:** Established a stable onset window (Oct 26 - Nov 4) consistent across a 5-year period.
